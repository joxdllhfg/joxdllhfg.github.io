2021-09-22

1.为什么说是初体验

首先，我以前接触电脑，都是玩游戏之类的日常娱乐，并没有去了解过关于计算机的知识与专业，再填报志愿是也是看到这个专业的人才稀缺，想着以后好就业，就报了这个专业。其次，在之前的9月17号的专业知识新老生交流会上，听了很多老师，学长学姐的介绍，我深刻了解了这个专业，知道了CTF这样的网络空间安全比赛，真真正正的走进了网络空间安全的世界。最后，我知道了网络空间安全又有许多方向，例如web,misc,pwn,crypto,reverse
等等，并且决定了自己要学习web的决心，对网络空间安全有了明确的方向。

接下来记录一下我这几天对C语言的自学的一下想法，因我还没有学习太多的内容，对C也还没有太深入的研究，所以接下来的想法都是一些粗鄙的个人见解。
1.C语言是一种功能很强的高级计算机语言（摘自《程序设计基础教程》），它让我了解到计算机真正的作用，可以说是逻辑的挑战。在这短短几天的学习后，我最深的感悟就是乔布斯所说的“I think everybody in this country should learn how to program a computer, should learn a computer language,because it teaches you how to think ”，我的思维方式已经有了变化，变得更加严谨，在以后的生活上也一定会受益匪浅。
